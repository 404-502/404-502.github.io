在Windows server上安装 SQL server 2008

2023/9/9 文章 SQL server

一、安装NAT3.5

在服务器管理器仪表板上选择添加角色和功能

一直点下一步到功能页面，并勾选 NET3.5 功能

然后等待安装完成

二、安装SQL server 2008

双击打开SQL server 2008安装包

选择全新安装或现有安装添加功能

然后一直默认点下一步或安装，直到下面这个页面，选择混合模式并设置管理员(sa)的密码

然后就一直下一步安装，等待安装结束

看到下面这个页面就是安装完成了

三、打开SQL server管理工具

在开始菜单栏里，找到并点击打开

选择SQL server身份认证登录，并输入登录名和密码，然后点击连接（登录名默认为sa）

成功进入SQL server
